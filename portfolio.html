LIRI Bot

LIRI Bot is a command line Node app that takes in parameters and returns specific data. Through the use of APIs, users can search Spotify for songs, Bands In Town for concerts, and OMDB for movies. This application employs the use of NPM packages, including Moment, Request, and the Node-Spotify-API package.

Bamazon

Bamazon is a command line Node app that uses MySQL intended to mimic Amazon's web service. There is a Customer View, Manager View and Supervisor view, giving a holistic view of an online sales business. This application employs the use of NPM packages, including MySQL, Inquirer, and the Cli-Table package.

Full-stack Projects
Friend Finder

Friend Finder is a compatibility-based application meant to simulate a simple dating app. This full-stack app takes in results from a user's response to questions and compares his or her answers with those from other users. The app will then display the name and picture of a user with the best overall match.

 Deployed  GitHub Repo
Eat-Da-Burger!

"Eat-Da-Burger!" is a Full-Stack restaurant app that allows user to input the names of burgers they'd like to eat. Burgers are kept in the waiting area until they are devoured, after which they are placed in the "Devoured" section. This full-stack app uses MySQL, Node, Express, and Handlebars to track a burger's status.

 Deployed  GitHub Repo
GlobeTrotter

GlobeTrotter is a Full-Stack group project. A user enters trip information including dates, destination and airlines. The results page provides the user with a suggested packing list, flight information, suggested hotels, and an activity planner. This app uses MySQL, Node, Express, Handlebars, and multiple APIs.
